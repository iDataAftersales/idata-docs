## 维护说明

### 更新文档

1. 将新 PDF/DOC 文件放入此目录
2. 编辑 `knowledge_base_catalog.json`，追加新记录
3. 双击运行 `sync.bat` 推送到 GitHub
4. Cloudflare Pages 1-2分钟内自动部署

### 文件结构

```
/
├── index.html                    # 下载页面主文件
├── knowledge_base_catalog.json   # 文档清单（JSON）
├── sync.bat                      # 一键同步脚本（Windows）
├── .gitignore
├── README.md
└── CN_iData_*.pdf / *.docx ...   # 所有文档文件
```
