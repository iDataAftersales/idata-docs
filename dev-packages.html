⚙️ 保存到 GitHub 设置
配置 GitHub Token 后，点击「💾 保存到 GitHub」即可直接将开发包目录提交到仓库，无需手动下载和上传。

如何获取 Token：
1. 打开 github.com/settings/tokens/new
2. 勾选 repo 权限（完整仓库访问）
3. 点击「Generate token」复制保存（以 ghp_ 开头）

离线备份：数据同时保存在浏览器本地，刷新不丢失。
